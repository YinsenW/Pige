1. Pige 是什么

Pige 是一个本地优先（Local-first）的通用知识 Agent，帮助你高效地处理、理解和沉淀各类知识。

  • 一次输入，自然生长
  • 文件是源，Markdown 是知识层
  • Agent 自主判断，减少人工干预

2. 核心理念

输入一次，Agent 自主理解、整理与沉淀，知识自然生长。
  • 相信 AI 与 Agent 能力会持续进化
  • 面向未来设计，减少阻断性确认，让体验更流畅
  • 知识沉淀在本地，安全可控，可持续积累

3. 核心能力

多格式解析PDF、Office、图片、网页等多种格式解析与理解
智能理解Agent 自主判断处理方式：回答 / 检索 / 解析 / OCR / 分析
知识沉淀生成结构化 Markdown，建立个人知识体系
高效检索本地全文检索，快速定位所需知识
安全本地所有数据保存在本地，隐私可控
