Pige UI 图标系统

通用系统动作采用在参考客户端构建产物中确认的开源 Lucide 图形语言，并从 24px 母版统一适配到桌面端 16–20px；Pige 品牌与产品专属语义保持独立。任何图标都必须先有明确功能，再进入界面。

母版24 × 24 Lucide 坐标；界面标准显示 20px
光学线宽母版 2px；20px 显示约 1.67px
端点round cap / round join；统一负空间与重心
语义设置使用齿轮，外观使用调色盘；图标不替代文字与可访问名称
